# Digital Watch - Linear Gradient Background

A sleek and modern **digital watch** built using **HTML, CSS, and JavaScript**.

This digital watch features:
- Eye-catching **linear gradient background**
- Clean and modern design with smooth interactions
- Fully responsive on desktop and mobile devices
- Displays hours, minutes, and seconds in real-time

This project is easy to customize and perfect for learning modern frontend design, UI/UX principles, gradient styling, and interactive elements.
